# Defamation AI

Defamation AI is a mobile assistant that helps users analyze text before publishing to detect potential defamation or reputation risks. It provides risk levels, explanations, and safer alternative wording.

## Features
- Risk detection (Low / Medium / High)
- Explanation of harmful content
- Safer rewritten suggestions

## Goal
Help users communicate responsibly and avoid legal or reputational harm.
